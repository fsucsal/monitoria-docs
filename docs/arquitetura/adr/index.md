# Decisões Técnicas de Arquitetura

Este documento centraliza todas as **Architecture Decision Records (ADRs)** do projeto.

As ADRs registram:

- Decisões arquiteturais
- Padrões técnicos
- Estratégias de segurança
- Infraestrutura
- Evolução futura do sistema

---

**Qualquer mudança relevante na arquitetura exige a criação de uma nova ADR.**
