# API v1 — Sistema de Seleção de Monitoria (UCSal)

Esta é a documentação oficial da API REST da plataforma de seleção de monitorias.

- Padrão: OpenAPI 3.0
- Autenticação: JWT (Bearer Token)
- Versão: v1

---

## Autenticação

Todos os endpoints protegidos utilizam:

Authorization: Bearer <seu_token>

Os tokens são obtidos via:

- `POST /auth/login`
- `POST /auth/refresh`
