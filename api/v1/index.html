    API - Sistema de Seleção de Monitoria (UCSal) 1.0.0

    API REST para inscrição, seleção, gestão e certificação de monitorias.

    Padrões importantes

    • Responses padronizadas
    • Erros tratados de forma centralizada
    • API como contrato (não expõe detalhes internos)

    Servers

    Description URL
    Produção (exemplo) https://api.ucsal.edu.br/api/v1
    Local http://localhost:8080/api/v1
